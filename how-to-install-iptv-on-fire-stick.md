## Step-by-Step: Install TiviMate on Fire Stick

### 1. Enable Apps from Unknown Sources

Your Fire Stick blocks sideloaded apps by default. You need to allow them first:

1. From the Fire Stick home screen, go to **Settings**
2. Select **My Fire TV** (or **Device** on older models)
3. Select **Developer Options**
4. Turn on **Apps from Unknown Sources**

> On Fire OS 7+, this setting is per-app. You'll enable it for the Downloader app in the next step.

### 2. Install the Downloader App

1. Go to the Fire Stick **Home Screen**
2. Select **Search** (magnifying glass icon)
3. Type **Downloader** and select it from the results
4. Click **Download** / **Get** to install it
5. If prompted, allow Downloader to install unknown apps

### 3. Install TiviMate via Downloader

1. Open the **Downloader** app
2. In the URL field, enter the TiviMate download URL (search "TiviMate APK" for the latest official link)
3. The APK file will download — click **Install** when prompted
4. Once installed, click **Done** (not Open — you'll open it from the home screen)

### 4. Add Your IPTV Subscription

1. Open **TiviMate** from your apps list
2. Select **Add Playlist**
3. Choose **Xtream Codes** as the playlist type
4. Enter your credentials:
   - **Server URL** — the address from your welcome email (e.g., `http://provider.com:port`)
   - **Username** — your IPTV username
   - **Password** — your IPTV password
5. Tap **Next** — TiviMate will connect and load your channel list

### 5. Start Watching

Your channels are now organized by category in the TiviMate interface. Use the EPG (program guide) to see what's on, set favorites, and browse on-demand content.

## Alternative: Using an M3U Playlist URL

If your provider gave you an M3U URL instead of Xtream Codes credentials:

1. In TiviMate, select **Add Playlist** → **M3U Playlist**
2. Choose **Enter URL**
3. Paste your full M3U playlist URL
4. Wait for the channel list to load

## Tips for the Best Fire Stick IPTV Experience

- **Use ethernet if possible.** The Amazon Ethernet Adapter eliminates Wi-Fi instability. This is the single biggest improvement for buffering issues.
- **Clear the cache regularly.** Go to Settings → Applications → Manage Installed Applications → TiviMate → Clear Cache.
- **Keep Fire OS updated.** System updates often improve streaming performance and network handling.
- **Close background apps.** Fire Stick has limited RAM — force-stop apps you're not using.
- **Use a VPN if your ISP throttles streaming.** Some ISPs reduce bandwidth for IPTV traffic. A VPN can bypass this.

## Troubleshooting

| Problem | Solution |
|---|---|
| "Unable to connect" error | Double-check server URL, username, and password. Make sure there are no extra spaces. |
| Channels load but won't play | Check your internet speed. Try a different channel — some may be temporarily offline. |
| App crashes on open | Clear cache and data, or reinstall the APK via Downloader. |
| EPG not showing | Go to TiviMate Settings → EPG → Update. Some providers update EPG on a schedule. |
